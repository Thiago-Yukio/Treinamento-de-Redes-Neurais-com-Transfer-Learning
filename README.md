# Treinamento-de-Redes-Neurais-com-Transfer-Learning

## Descrição:
Projeto de Deep Learning utilizando **Transfer Learning com MobileNetV2** para classificar imagens de **gatos e cachorros**, desenvolvido em **Python no Google Colab**.

---

## Recursos:
- Python  
- TensorFlow / Keras  
- Google Colab

---

## Como funciona:
- Pré-processamento de imagens (normalização e augmentation)
- Uso de modelo pré-treinado (MobileNetV2)
- Adição de camadas finais para classificação
- Treinamento e avaliação do modelo

---

## Conclusão:
O projeto demonstra, de forma prática, como usar modelos pré-treinados para resolver problemas de classificação de imagens.

---

OBS: Foi utilizada uma imagem "gato esquisito" para verificar a capacidade de **generalização** do modelo em casos fora do padrão.
